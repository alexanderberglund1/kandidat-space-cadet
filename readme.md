# Space Cadet

Space Cadet är en interaktiv 2D-rymdsimulator skriven i Python med Pygame.  
Du kan skapa egna planeter, se hur de påverkar varandra med gravitation och följa deras banor i realtid.

---

## Funktioner

- Newtonsk gravitation mellan alla kroppar (N-body).
- Central stjärna som håller ihop systemet.
- Planeter som kan:
  - skapas med musen,
  - kollidera och slås ihop,
  - slungas iväg ut ur systemet.
- Trails som visar banorna över tid.
- Zoom och kamerapanorering.
- En inbyggd demo som startar ett färdigt litet “solsystem”.
